# DL Enterprises – Invoice Management System
### React + Supabase | Production Ready

**Quick Start:**
1. Run SQL from `supabase_setup.sql` in Supabase SQL Editor
2. Add your keys to `src/supabase/config.js`
3. `npm install && npm start`
4. Deploy: `vercel --prod`

See **SETUP_GUIDE.md** for full step-by-step instructions.
